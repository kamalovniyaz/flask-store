## Проект "Интернет-магазин на Flask и SQLite"
Привет! Добро пожаловать в репозиторий проекта интернет-магазина, разработанного с использованием фреймворка Flask и базы данных SQLite. Этот проект представляет собой пример простой системы интернет-магазина, где вы можете добавлять, просматривать и покупать товары.

Описание проекта
Данный проект реализован на языке программирования Python с использованием веб-фреймворка Flask и базы данных SQLite. Основная идея заключается в создании интернет-магазина, который позволяет пользователям просматривать список товаров, добавлять их в корзину и оформлять заказы.
